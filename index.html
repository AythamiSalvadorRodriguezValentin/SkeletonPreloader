#GetStarted

How to use: 

Add to index.html into head

<script type="text/javascript" src="https://goo.gl/Ct7zVn" />
<link type="text/css" src="https://goo.gl/dWVRt6" />

Inner into your container:

<div class="pr-container">
   <!-- This element will show during the load -->
   <div class="pr-on"></div>
   <!-- This element will show when the loading proces will be done -->
   <div class="pr-off></div>
</div>

Run in your file.js

var preloader = new SkeletonPreloader();

Suggestion of use (JS):

API.request.then(res => { 
//Run your code with the promise;
 preloader.finish()
})
